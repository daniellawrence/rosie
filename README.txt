Rosie the Robot
---------------

Tracks all of our deploys, so we do need too.


deploying
----------

	./fab.py -R systest deploy

Then open up http://rosie.f2l.info/

    
Running locally
---------------

	mkvirtualenv rosie
	pip install -U -r requirements.txt
	./main.py

Then just open up http://localhost:5000/ on your machine!
